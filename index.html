Sobre mi

Soy desarrollador web con experiencia en front-end y back-end además cuento con conocimientos en diseño web. Mi experiencia radica en la creación de paginas web informativas que fácilitan la interacción con el usuario. Dentro de mis habilidades como desarrollador se encuentran el manejo de PHP puro, HTML, CSS, JavaScript Y Python. A traves de mi trabajo busco crear soluciones a empresas a fin de obtener una mejor comunicación con el cliente final mediante portales y paginas web.
